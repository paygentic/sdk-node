# InvoiceLineItemLineItemType

Type of line item: 'charge' for regular billing, 'refund' for refunded items (amounts are negated)

## Example Usage

```typescript
import { InvoiceLineItemLineItemType } from "@paygentic/sdk/models";

let value: InvoiceLineItemLineItemType = "charge";
```

## Values

```typescript
"charge" | "refund"
```
