# InvoicesV2

## Overview

Invoice V2 operations supporting billing cycles organized by time periods. Warning: v0 invoice endpoints are no longer supported.

### Available Operations

* [listInvoices](#listinvoices) - List
* [getInvoice](#getinvoice) - Get
* [getInvoiceLineItems](#getinvoicelineitems) - Get Line Items

## listInvoices

List invoices with optional filters. Platform users can use nextActionAt=ready to get invoices ready for processing.

### Example Usage

<!-- UsageSnippet language="typescript" operationID="listInvoices" method="get" path="/v2/invoices" -->
```typescript
import { Paygentic } from "@paygentic/sdk";

const paygentic = new Paygentic({
  bearerAuth: process.env["PAYGENTIC_BEARER_AUTH"] ?? "",
});

async function run() {
  const result = await paygentic.invoicesV2.listInvoices({});

  console.log(result);
}

run();
```
