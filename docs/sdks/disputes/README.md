# Disputes

## Overview

A `Dispute` enables customers to contest usage events that they consider to be inaccurately recorded or billed.

### Available Operations

* [createDispute](#createdispute) - Create
* [listDisputes](#listdisputes) - List

## createDispute

Create a dispute for a usage event. Only one dispute can be created per usage event.

### Example Usage

<!-- UsageSnippet language="typescript" operationID="createDispute" method="post" path="/v0/disputes" -->
```typescript
import { Paygentic } from "@paygentic/sdk";

const paygentic = new Paygentic({
  bearerAuth: process.env["PAYGENTIC_BEARER_AUTH"] ?? "",
});

async function run() {
  const result = await paygentic.disputes.createDispute({
    message: "<value>",
    usageEventId: "<id>",
  });

  console.log(result);
}

run();
```
